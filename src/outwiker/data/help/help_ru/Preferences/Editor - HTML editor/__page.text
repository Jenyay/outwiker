!!! Редактор - HTML-редактор

%center%Attach:outwiker_opt_editor_html.png

Данный раздел содержит настройки для раскраски HTML-кода на [[HTML-странице -> page://__7bd312fb-c3e4-4d9d-9267-cc394dbb9f20]] или на вкладке ''HTML'' [[викистраницы -> page://__19866d72-8d95-4e2c-8df1-6df0bd8f06c0]]. Здесь можно задать цвета и начертание шрифта для HTML-тегов, их атрибутов, чисел, HTML-комментариев, строк в формате HTML, а также ошибочных HTML-элементов.

Имя шрифта и его размер устанавливаются в разделе [[Редактор - Общее -> page://settings_editor_general]].
