Поиск в программе реализован в виде отдельного типа страниц, которые могут располагаться в дереве где угодно. Создавать страницу поиска можно точно так же, как и другие страницы, но есть и упрощенный способ создания, который состоит в том, что при первом нажатии соответствующей кнопки или пункта меню «Инструменты – Глобальный поиск…», будет создана и сразу выбрана страница поиска с именем «# Поиск». Если страница поиска с именем «# Поиск» уже есть, то при выборе этого пункта меню программа переключится на эту страницу.

Такое представление поиска может быть удобно, если вы часто ищите одни и те же фразы. Теперь можно создать несколько страниц для каждого поиска и программа будет запоминать что именно вы искали с каждой такой страницы. 

Искать можно как по содержимому страницы, так и по тегам. Если не вводить искомую фразу, то поиск будет осуществляться только по тегам, причем можно выбирать, страница должна содержать все отмеченные теги или хотя бы один из них. 

Если никакие теги не отмечены, то при поиске теги не будут учитываться. Если оставить пустым поле для искомой фразы и не отметить ни одного тега, то будут выведены все страницы в дереве.
