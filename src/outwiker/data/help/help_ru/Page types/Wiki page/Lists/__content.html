Общие сведения


С помощью викинотации можно создавать нумерованные и ненумерованные списки. Для создания ненумерованных списков используется один или несколько символов "*", стоящие в начале строки. Для создания нумерованных списков используется один или несколько символов "#", стоящие в начале строки. Чтобы создать сложенный список, символы "*" или "#" нужно повторить столько раз, каким должен быть уровень вложенности.

Для создания ненумерованного списка можно воспользоваться пунктом меню "Вики - Списки - Ненумерованный список" или кнопкой .

Для создания нумерованного списка можно воспользоваться пунктом меню "Вики - Списки - Нумерованный список" или кнопкой .

Если курсор стоит на строке, которая уже является строкой списка (строка начинается с символа "*" или "#"), то нажатие кнопок и (или выбор аналогичного пункта меню) увеличивает уровень вложенности списка.

Далее приведены несколько примеров использования списков в викинотации.

Ненумерованные списки

Синтаксис



* Строка 1
* Строка 2
** Строка 3
** Строка 4
*** Строка 5
*** Строка 6
** Строка 7
*Строка 8

Результат

  • Строка 1
  • Строка 2
    • Строка 3
    • Строка 4
      • Строка 5
      • Строка 6
    • Строка 7
  • Строка 8

Нумерованные списки

Синтаксис



# Строка 1
# Строка 2
## Строка 3
## Строка 4
### Строка 5
### Строка 6
## Строка 7
# Строка 8

Результат

  1. Строка 1
  2. Строка 2
    1. Строка 3
    2. Строка 4
      1. Строка 5
      2. Строка 6
    3. Строка 7
  3. Строка 8

Смешанные списки

Синтаксис



# Строка 1
# Строка 2
** Строка 3
** Строка 4
### Строка 5
### Строка 6
** Строка 7
# Строка 8

Результат

  1. Строка 1
  2. Строка 2
    • Строка 3
    • Строка 4
      1. Строка 5
      2. Строка 6
    • Строка 7
  3. Строка 8

Стили списков


Для ненумерованных списков можно изменять стиль буллетов (так в издательском деле называют значки, которые стоят в начале каждого пункта списка). Это может быть полезно, например, для оформления списа дел, тогда предстоящие дела можно помечать одним стилем, а выполненные дела - другим (для этого предусмотрены специальные стили). Стили списков полезны, если вы используете методологию ведения списков Bullet Journal.

Стиль указывается после символа "*" ненумерованного списка и представляет собой комбинацию квадратных скобок, между которыми может быть дополнительный символ, определяющий стиль. В следующей таблице перечислены поддерживаемые на данный момент стили:
