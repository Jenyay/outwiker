Программа outwiker может принимать параметры командной строки. Формат аргументов следующий:

[>[@outwiker [--page PAGE] [--help] [--version] [--readonly] [--normal] [--debug] [--disable-plugins] [path]@]<]

'''path''' - путь до дерева заметок. Например: 

[>outwiker.exe "C:\temp\samplewiki"

outwiker.exe "C:\temp\samplewiki\__page.opt"<]

'''-p, --page''' - UID или относительный путь до страницы, которую нужно открыть при запуске OutWiker. Например:

[>outwiker.exe --page="Страница 1/Страница 2/Страница 3" "C:\temp\samplewiki"

outwiker.exe -p "Страница 1/Страница 2/Страница 3" "C:\temp\samplewiki"

outwiker.exe --page="[=page://__1640eea0-1bb0-453b-ba4c-07f1c298ce76=]" "C:\temp\samplewiki"
<]

'''-h, --help''' - Показать справку по параметрам командной строки.

'''-v, --version''' - Показать версию OutWiker.

'''-r, --readonly''' - Открыть дерево заметок в режиме "только для чтения".

'''--normal''' - Не минимизировать главное окно OutWiker при запуске.

'''--debug''' - Запуск в режиме отладки.

'''--disable-plugins''' - Не загружать установленные плагины.
