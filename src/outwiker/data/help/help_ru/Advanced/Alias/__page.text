Заголовок страницы, который отображается в дереве заметок, не всегда совпадает с именем папки, которая содержит файлы этой страницы. Например, файловые системы имеют ограничение на то, какие символы можно использовать в именах файлов и папок, например, в Windows запрещены такие символы как *, ?, :, /, \ и др. Но эти символы иногда хотелось бы использовать в именах страниц. Чтобы обойти это ограничение, у страницы может быть установлен псевдоним, и при его наличии именно он будет отображаться в дереве заметок. 

Псевдоним устанавливается автоматически при создании страницы с запрещенными символами, а также, если создается страница с именем, которое уже существует на том же уровне вложенности. В этом случае имя папки будет скорректировано таким образом, чтобы удовлетворить требованиям файловой системы, но для пользователя заметка будет отображаться с тем именем, которое он ввел.

Псевдоним задается в файле настроек заметки __page.opt в разделе [General] с помощью параметра alias.

В основном интерфейсе OutWiker не предусмотрена возможность непосредственно влиять на этот параметр, поскольку пользователям обычно не так важно, совпадает имя папки заметки с именем заметки или нет. Однако, если такая возможность вам требуется, можно воспользоваться плагином [[HackPage -> https://jenyay.net/Outwiker/HackPage]], который дает возможность устанавливать и изменять псевдонимы страниц.

При задании [[ссылок -> page://wiki_links]] на другие страницы можно использовать как псевдонимы страниц, так и имена папок, соответствующие страницам, однако рекомендуется создавать ссылки с использованием идентификаторов.
