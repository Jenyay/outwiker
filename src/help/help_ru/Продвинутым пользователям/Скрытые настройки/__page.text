В этом разделе перечисляются настройки, которые нельзя поменять через интерфейс программы, но можно исправлять в файле outwiker.ini. Если какого-то из указанных параметов нет в файле outwiker.ini, его можно добавить.

* '''[MainWindow] / datetime_status_width''' - Целочисленное значение. Определяет ширину области в статусной панели, выделенной для показа даты последней правки страницы. Значение по умолчанию = 250.
