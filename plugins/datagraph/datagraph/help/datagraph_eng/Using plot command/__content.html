Команда plot


Команда plot предназначена для построения графиков в осях X и Y. Общий формат использования этой команды выглядит следующим образом:

(:plot <параметры>:)
    <данные>
(:plotend:)


С помощью параметров можно указать, откуда команда будет брать данные, задавать формат календарных данных, а также менять оформление графика.

Например, команда

(:plot
    curve.data="Attach:example.txt"
:)
(:plotend:)


создает следующий график по данным прикрепленного файла example.txt
